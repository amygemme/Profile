Online Store Inventory Tracker

The database stores data for items to be sold in an online store. The data includes price, department and quantity in stock. The code will first retrieve a list of departments for the user to shop in, followed by a list of items in the selected department, followed by the price of the selected item. The user can then choose to purchase or not. If the user choses to purchase, the database will update the inventory appropriately.
